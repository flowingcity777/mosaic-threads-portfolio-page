## ✨ About the Project

Mosaic Threads explores how digital tools can translate meaningful images into **physical, handmade creations**.

The tool allows users to:
- Upload an image
- Convert it into a limited yarn color palette
- Generate a stitch-based knitting chart
- Download the pattern for offline use

---

## 🧠 UX Focus

This case study highlights:

- Designing for **real-world constraints** (yarn, stitches, limited colors)
- Prioritizing **usability over visual perfection**
- Iterating from noisy outputs to **clear, knittable patterns**

> A usable pattern is better than a perfect image.
